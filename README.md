# 🚀 Cursor Website Clone

A frontend clone of the Cursor website built using HTML and CSS.  
This project replicates key layout sections including the sticky navbar, hero section, trusted-by logos, and recent highlights section with sticky sidebar behavior.

---

## 📌 Features

- ✅ Sticky navigation bar
- ✅ Clean and structured CSS
- ✅ Reusable CSS variables
- ✅ Modern UI styling

---

## 🛠️ Built With

- HTML5
- CSS3
- Flexbox
- CSS Variables
- Font Awesome (for icons)

---

## 📂 Project Structure

```
project-folder/

├── index.html
├── style.css

│   └── images, icons
└── README.md
```

## 🎯 Sections Implemented

### 1️⃣ Navbar

- Sticky positioning
- Centered content
- Responsive layout
- Proper z-index handling

### 2️⃣ Hero Section

- Main heading
- Subtext
- Call-to-action buttons

### 3️⃣ Trusted By Section

- Grid-based logo layout
- Clean card design
- SVG logos support

### 4️⃣ Recent Highlights

- Two-column layout
- Sticky left heading
- Scrollable right content
- Stops sticking when section ends

---

## 🧠 Key Concepts Practiced

- `position: sticky`
- `z-index` stacking context
- Flexbox alignment
- Sticky inside flex containers
- Avoiding overflow issues
